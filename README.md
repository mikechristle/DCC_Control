# DCC_Control
A project that controls a DCC model train set.
## Python Source for the Pico
The python source files that run on the Raspberry PI Pico to control the DCC signal
## Python Source for the PC Control Program
THe python source files that run on a PC to communicate with the Raspberry PI Pico.
## KiCad PCB Files
KiCad source files for the printed circuit board.
## Sources for the DCC_Monitor application
Sources for a seperate project that will monitor and display the contents of the DCC signal.
